用意

本系列博客用于期末复习Matlab，请知悉。这也就意味着本系列文章均为基础入门内容，也有可能存在些许错误。

Matlab运算符

  • 括号： 小括号代表运算级别，中括号用于生成向量和矩阵，花括号用于生成单元数组（cell）。
  • * 与 .* ，前者表示数组相乘，后者表示点乘。类似有 / ./ ; \ .\ ; ^ .^ 。

基础操作命令

  • clc （clear command）清空命令窗口而不清空变量
  • clear 清空工作区变量而不清空窗口命令
  • clf （clear figure）清空图形窗口
  • disp （display）显示文字或者变量内容
  • hold 保持图形
  • load 加载指定文件变量
  • type 显示文件内容

其它

变量名必须以字母开始，其后可以为字母、数字或者下划线。
