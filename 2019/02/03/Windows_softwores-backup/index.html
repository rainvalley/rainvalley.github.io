Windows软件备份

Feb 3, 2019

QWQ,不久前实在受不了Windows睡眠/休眠无法唤醒，联系了Dell售后，一致表示要重装系统。好吧，我把系统重置了，留下这个备份单。

PS:一些例如Chrome，TIM，网易云什么的，用到自然会下的QWQ，也就。。。不写了吧（逃

Hash Tab，存在感极低的校验文件，可以在文件属性中显示CRC32，MD5 ,SHA-1的Hash值。

Ubuntu，MS为Windows编译了Linux内核，然后就。。。奇妙技法（WSL)。

Vlc，让我感动的跨平台播放器。

WinSCP，该介绍什么？

Todolist，懒癌晚期救星，App也是全平台的。

QPST，高通放出的用来调试自己家Soc的东西，当然，要进入9008模式。

Sd formatter，用于SD卡的格式化。

7-zip，和HashTab一样，存在感极低，但又特别的好用。

Dell PremierColor，不多说，XPS 15@4K自带的专业调色软件。

TeamViewer，远程桌面，要知道有些Windows版本是把远程桌面给移除了。

Pan Download，好用的第三方百度云下载器，成功把我100M的宽带跑满了。当然，这是在氪金的前提下，话说，百度云为什么连VIP也要限速。所以说，百度💊。

Putty，什么SSH连接，本地串口，都能用它。

FinalShell，好用的终端连接工具，功能齐全，直观，只是不知道安全性怎么样。

VS Code，不解释，好用的IDE.

rufuns，常用来刻录iso镜像。

Miflash,这，加上吧。
