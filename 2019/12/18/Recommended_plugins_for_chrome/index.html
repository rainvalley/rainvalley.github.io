Chrome插件推荐

Dec 18, 2019

在Chrome浏览器地址栏输入chrome://extensions/即可打开扩展程序管理器。
我正在使用的插件如下：

依据使用频率排序：

  • Vimium：把Vim的操作方式带给Chrome，掌握了Vim之后很容易上手。如同Vim一般，插件拥有Normal，Command-Line，Insert，Visual模式（也许会写一篇文件介绍Vim的使用。）常用键码如下：
键码 用途
k/j/h/l 上/下/左/右滑动网页
u/d 上/下翻半页
o/O 当前/新标签页打开网页
J/K 左/右移动标签页
H/L 向前/后移动网页
p/P 当前/新标签页访问剪切板内容
x/X 关闭当前标签页/打开最近被关闭的标签页

  • SimpRead：一个提供阅读模式的插件，能够最大限度简化页面元素，提供良好阅读体验，使用快捷键aa即可进入阅读模式。

  • 沙拉查词：界面大致如下，提供了丰富的的词库，光标划词便可进行单词或整句翻译。

  • Picture-in-Picture：看名字便可知是画中画功能，与Android上的画中画类似，开启之后可以提供一个类似悬浮窗的视频播放页面（划水必备），大概就像这样了：
  • 广告终结者：这个似乎是用的最多的一个插件，无奈存在感太低，不过这或许是一件好事？
  • Google环聊：集成环聊服务与Google Voice，用来收国内垃圾短信还是很好用的：
  • 购物党自动比价工具：避免被js坑的必备插件，可以查看最近一年该商品的价格。最重要的是，该插件有微信推送功能，当全网最低价格低于订阅价格便推送给订阅者微信。
  • Wappalyzer：可以嗅探当前网站使用的前端技术，诸如js库，字体渲染库，服务器类型等等等。
  • Listen1：集合并可供播放多个平台的音频资源。
  • 超星慕课小工具：挂科必备，经常用来刷网课，例如智慧树，学习通。PS：该插件使用时需要保持该网页在浏览器前台，即不可切换至其他标签页，否则容易失效。
